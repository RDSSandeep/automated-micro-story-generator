# Automated Micro Story Generator

A CLI application that generates short micro stories from three keywords: **character**, **place**, and **object**. Supports both template-based generation (V1) and AI-powered generation via the Anthropic Claude API (V2).

**Team:** Dhruv Sharma, Durga Sai Sandeep Rayapureddy, Isit Pokharel, Tan Nguyen

---

## Features

- **V1 (Template mode):** Multi-paragraph stories built from genre-specific templates (adventure, mystery, fantasy, sci-fi, comedy)
- **V2 (AI mode):** Creative stories generated by Claude, with automatic fallback to templates if the API is unavailable

---

## Setup

```bash
git clone <repo-url>
cd automated-micro-story-generator
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# For V2 AI mode: copy .env.example to .env and add your API key
cp .env.example .env
# Edit .env and set ANTHROPIC_API_KEY=your-key-here
```

---

## Running V1 (Template Mode)

No API key required.

```bash
python run.py
```

Choose `t` (Template), enter three comma-separated keywords, pick a genre (or "random").

---

## Running V2 (AI Mode)

Requires `ANTHROPIC_API_KEY` in `.env`. Falls back to template mode if the API is unavailable.

```bash
python run.py
```

Choose `ai` (AI-Powered), enter keywords and genre.
